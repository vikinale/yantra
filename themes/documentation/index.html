Introduction

Flapt Admin is a popular open source WebApp template for admin dashboards and control panels. It is built with Bootstrap. If you are unfamiliar with Bootstrap, visit their website and read through the documentation. All of Bootstrap components have been modified to fit the style of Flapt Admin and provide a consistent look throughout the template. It is a responsive HTML template that is based on the CSS framework Bootstrap It utilizes all of the Bootstrap components in its design and re-styles many commonly used plugins to create a consistent design that can be used as a user interface for backend applications. Flapt Admin is based on a modular design, which allows it to be easily customized and built upon. This documentation will guide you through installing the template and exploring the various components that are bundled with the template.
